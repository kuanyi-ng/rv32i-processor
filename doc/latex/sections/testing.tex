  今回設計したプロセッサの機能検証を行うために, 
  表 \ref{table:test-program} にあるプログラムが用意された.

  論理シミュレーター xmverilog と
  波形ツール SimVision を用いて, シミュレーションを行った.
  テストプログラムが正しく実行され、正確な出力が得られたことを確認した.
